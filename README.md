# Checador-De-Senha
Script em Python para checar se sua senha é segura!

Requisitos
-----------
Python 3
Linux ou Mac Os

Modo de uso
-----------
Basta executar no terminal o comando:
    python checador.py

Orientações
-----------
formas de commts: se você fez qualquer alteração, commita e manda, explicando
com suas paralvras o que foi feito.
